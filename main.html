Question-3: What is event delegation in the DOM?

Event delegation in JavaScript used to handle events on elements, by it we can add event listener to a parent element of the target elements, When an event is triggered,it bubbles up, and the parent element can handle the event for its children.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in JavaScript's, where an event starts at the target element and then "bubbles up" through the dom tree to its parents, triggered any event listeners added to those parent elements.
